## 📂 Featured Projects

### 🔴 Financial Fraud Detection System
> Real-time fraud detection pipeline on 6.3M+ transactions

- **Stack:** Python · PySpark · Apache Kafka · Docker Compose
- Built Kafka streaming + PySpark Structured Streaming pipeline on **6.3M transactions** (0.13% fraud rate)
- Applied **SMOTE oversampling** to handle class imbalance; evaluated with ROC-AUC, Precision-Recall, F1
- Designed feature engineering pipeline with EDA; deployed full stack via **Docker Compose** with alerting

---

### 🍽️ DealRadar Restaurant Discount Aggregator &nbsp;`97/100`
> AI-powered bank card deal & restaurant discount platform

- **Stack:** Python · BeautifulSoup · Neon PostgreSQL · KNIME · n8n · Streamlit
- Scraped live offers with BeautifulSoup; stored in **Neon PostgreSQL** (cloud-native)
- Automated pipeline via **KNIME** preprocessing + **n8n Cloud** workflow automation
- Integrated **RadarBot AI chatbot** using Gemini 2.5 Flash for natural language deal queries
- Deployed interactive dashboard on **Streamlit Cloud**

---

### 🏥 MediExplain AI Diabetes Edition
> Multi-agent NLP pipeline for medical lab report interpretation

- **Stack:** Python · Groq API (LLaMA-3.3-70B) · Streamlit · Docker
- Built **multi-agent pipeline**: symptom extraction → RAG-based drug knowledge → PDF report generation
- Achieved **health score 80/100** on random patient blood test with correct abnormality detection
- Deployed via **Docker** and GitHub; used fpdf2 and ReportLab for PDF generation

---

### 🧠 Deep NLP Sentiment Analysis Benchmark
> Comprehensive benchmark of 10 NLP architectures

- **Stack:** Python · TensorFlow · BERT · DistilBERT · RoBERTa · T5
- Benchmarked **10 models**: RNN, LSTM, GRU, BiLSTM, BiGRU, TextCNN, Transformer, BERT, DistilBERT, RoBERTa
- Dataset: **Flipkart product reviews** — evaluated on Accuracy, Precision, Recall, F1, training time
- Best performance achieved with **RoBERTa**; full comparative report with training curves
