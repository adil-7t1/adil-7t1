## 📂 Featured Projects

### 🤖 [Aria AI WhatsApp Personal Assistant](https://github.com/adil-7t1/Aria_Ai)
> Intelligent WhatsApp assistant with voice, image & web search capabilities

- **Stack:** n8n Cloud · Groq API · Meta WhatsApp Business API · Tavily Search
- Handles **text, voice, and image** messages via Switch node routing
- Voice transcription with **Groq Whisper Large V3**
- Image analysis with **LLaMA 4 Scout Vision**
- Web search integration with **Tavily API** + conversation memory

---

### 🔴 [Financial Fraud Detection System](https://github.com/adil-7t1/fraud-detection-bda)
> Real-time fraud detection pipeline on 6.3M+ transactions

- **Stack:** Python · PySpark 3.5.1 · Apache Kafka · Docker Compose
- Built Kafka streaming + PySpark Structured Streaming on **6.3M transactions** (0.13% fraud rate)
- Applied **SMOTE oversampling** to handle extreme class imbalance
- Evaluated with ROC-AUC, Precision-Recall, F1; full EDA with 4+ visualizations

---

### 🍽️ [DealRadar Restaurant Discount Aggregator](https://github.com/adil-7t1/DealRadar)
> AI-powered bank card deal & restaurant discount platform

- **Stack:** Python · BeautifulSoup · Neon PostgreSQL · KNIME · n8n · Streamlit
- Scraped live offers from 3 sources; stored in **Neon PostgreSQL** (cloud-native)
- Automated pipeline via **KNIME** preprocessing + **n8n Cloud** workflow automation
- Integrated **RadarBot AI chatbot** using Gemini 2.5 Flash
- Deployed interactive dashboard on **Streamlit Cloud**

---

### 🏥 [MediExplain AI Diabetes Edition](https://github.com/adil-7t1/MediExplain-AI)
> Multi-agent NLP pipeline for medical lab report interpretation

- **Stack:** Python · Groq API (LLaMA-3.3-70B) · Streamlit · Docker · FAISS
- Built **6-agent pipeline**: PDF extraction → NLP → RAG → Abnormality → Drug check → LLM report
- Achieved **health score 80/100** on random patient blood test
- Deployed via **Docker** with PDF report generation (fpdf2 + ReportLab)

---

### 🧠 Deep NLP Sentiment Analysis Benchmark
> Comprehensive benchmark of 10 NLP architectures on Flipkart reviews

- **Stack:** Python · TensorFlow · BERT · DistilBERT · RoBERTa · T5
- Benchmarked **10 models**: RNN → LSTM → GRU → BiLSTM → TextCNN → BERT → RoBERTa
- Full comparative report with Accuracy, Precision, Recall, F1, training time
- **RoBERTa** achieved best performance
